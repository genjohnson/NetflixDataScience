What is Singular Value Decomposition?

Refer to slides from David Stewart: https://icon.uiowa.edu/d2l/le/content/2456868/viewContent/4420390/View

Created a matrix, where each row corresponds to a user each column corresponds to a movie. This is a sparce matrix (A), since there are 1700 movies, and each user has rated relitively few of them. We then adjusted each of the ratings to account for user bias and movies that consistently have extreme ratings. This gave us matrix B. Some users really like movies in general, and tend to give them higher ratings on average, so we accounted for this by calculating the average rating each user provided and subtracting this from each of their ratings. Then from each of these adjusted ratings we subtracted the average rating of that movie provided by all users. Since we are working with such a large, sparce matrix, we needed to make a choice for the value of k to pass to the SVD(S) function. k corresponds to the number of characteristics of a movie to take into account when creating the recommendations, for example genre, lead actors, etc. [Add explanation of how we chose which value of k to use -- trial and visual inspection of results] Then we applied SVD to matrix B and chose a user to make recommendations for.
